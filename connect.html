Miner Configuration

This is a basic guide on how to setup your miner to our pool.

Mine with

AMD GPUs

Using

lolMiner
  • lolMiner [AMD][DevFee 1.5%] | Download Link

    lolMiner.exe --algo AUTOLYKOS2 --pool stratum+tcp://ergo.miningcrypto.live:3194 --user ERGO_WALLET_ADDRESS.WORKER_NAME

  • SRBMiner-Multi [AMD] [DevFee 2.0%] | Download Link

    SRBMiner-MULTI.exe --disable-cpu --algorithm autolykos2 --pool stratum+tcp://ergo.miningcrypto.live:3194 --wallet ERGO_WALLET_ADDRESS --password WORKER_NAME
